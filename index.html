    About Me

    Hi there! I am Ronda, a two-year-old Labrador Retriever who started training to become a guide dog for the blind when I was little.

    I am very friendly and energetic puppy, always eager to learn new things and meet new people and receiving new toys. With my intelligence, loyalty, and gentle nature (and cuteness of course),I have all the traits necessary to become a fantastic guide dog. This is why when I was one-year-old I passed all the tests to become a guide dog. Actually, I did so well that they decided they want to keep my genes, so my puppies will be guide dogs as well and provide invaluable assistance to those with visual impairments. So now, I am a mother of 9 little puppies who are undergoing their training.

    Q&A answers

    א-- הבדלים עיקריים בין web1, web2 ו-web3 הם:
    1. Web 1.0 הוא סטטי ומאפשר קריאה בלבד, כלומר היה ניתן לצפות במידע בלבד. Web 2.0 מאפשר קריאה וכתיבה, כלומר מאפשר אינטראקטיביות וכתיבת מידע. Web 3.0 מאפשר קריאה, כתיבה והרצה, כלומר אינטראקטיביות רבה ויצירת תוכנות להוספת מידע.
    2. Web 1.0 הוא בעל גרפיקה בסיסית וחווית משתמש עבור מערכות שולחניות. Web 2.0 הוא בעל עיצוב תגובתי ונגיש יותר, הכולל גם וידאו ועוד, ובעל חווית משתמש נוחה יותר ומזמינה. Web 3.0 בעל גרפיקה מתקדמת עוד יותר, ובעל עיצוב מותאם אישי למשתמשים, ותכונות חדישות כמו תלת מימד.
    3. ב Web 1.0 יצירת התוכן היא על ידי מספר מסוים של ארגונים וחברות (או אנשים), בעוד שבWeb 2.0 נוצרו פלטפורמות שמאפשרות יצירת תוכן על ידי כלל המשתמשים ושיתופו, כך שכולם חשופים לכך. ב Web 3.0 כבר נוצרו מעבר לפלטפורמות גם תוכנות וחידושים כך שאנשים יוכלו ליצור בעזרתם עוד תוכן וכלים חדשים לחשוף אותו.
    ב-- דוגמאות:
    1. דוגמה הממחישה את ההבדל הראשון היא חיפוש מידע אודות כלבים נוספים באיזור שלי. ב Web 1.0 נוכל רק להגיע לאתר מסוים ולקרוא את הפרטים אודות האתר ולבדוק האם יש כלבים נוספים באיזור. ב Web 2.0 בנוסף לקריאה, נוכל גם לכתוב ולשאול שאלה שכזאת בפורום או להגיד לאנשים ליצור איתי קשר. ב Web 3.0 נוכל בנוסף לקריאה ולכתיבת בקשה משלנו, גם להשתמש בכלים ותוכנות, כמו למשל להשתמש בפייסבוק ולפנות לאנשים שמעלים תמונות עם כלבים וגרים באיזור.
    2. דוגמה הממחישה את ההבדל השני היא חלק יצירת הקשר באתרים. ב Web 1.0 בחלק זה של האתר פשוט יופיע מספר טלפון או אימייל שאיתו ניתן ליצור קשר עם כותב האתר ללא שום עיצוב מיוחד. ב Web 2.0 יופיע חלק יצירת קשר המאפשר שליחת הודעה לכותב האתר, באמצעי נוח ועיצוב גרפי מזמין, ב Web 3.0 חלק זה יכול להיות כדוגמת צ'אט בוט מותאם אישית העונה לך ואיתו יוצרים קשר או בצורה נוחה ואישית יותר.
    3. דוגמה הממחישה את ההבדל השלישי היא קריאת מידע אודות כלבי נחייה. ב Web 1.0 נוכל רק לקרוא מידע שנכתב על ידי ארגונים שעסקו בכך (למשל CNN) ב Web 2.0 נוכל לקרוא מידע שגם אחרים כתבו ופרסמו (למשל בעל כלב נחייה שפרסם סרטון ביוטיוב על כך, או דף ויקיפדיה שאנשים ערכו על הנושא). ב Web 3.0 נוכל כבר לקרוא מידע שאנשים כתבו ופרסמו במקומות שונים כך שגם אם אתקל בדף פייסבוק של מישהו לדוגמה אצליח ללמוד מידע אודות כלבי נחייה ולא רק אם אחפש אקטיבית מידע אודות כלבי נחייה.
